* **摄像头预览**
    摄像头数据预览大体思路：绘制长方形，将摄像头数据转成纹理，绘制在长方形上。
    长方形绘制需要四个顶点，绘制方式：GL_TRIANGLE_STRIP
    1. 顶点数据
    ```
    @Override
    protected void initVert() {
        float[] verts = {
                -w / 2, h / 2, 0,//左上
                w / 2, h / 2, 0,//右上
                -w / 2, -h / 2, 0,//左下
                w / 2, -h / 2, 0//右下
        };
        mVertSize = verts.length / 3;
        mVertBuf = ShaderUtils.getFloatBuffer(verts);
    }
    ```
